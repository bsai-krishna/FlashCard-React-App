# Flashcards

## About

Project built for study purposes.

## Use

To use this app:

1. Clone the repository:

```
git clone
```

2. Change directory into the repository's working directory:

```
cd flashcards
```

3. Install dependencies via npm:

```
npm install
```

4. Run the app by using:

```
npm start
```
